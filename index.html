Stacked trinket charts

Please notice that these values are all generated using the highest (read: difficulty level) available SimulationCraft profiles which are currently the T20M profiles. Each trinket was simulated without a second one. The profiles already contain two legendaries, so please be aware that the legendary trinket would be a third legendary in most cases, which isn't realistic. Furthermore all limitations applying to SimulationCraft apply to these graphs as well. This means that the accuracy of these graphs depends on the quality of the maintained action priority list (APL). A good APL does not necessarily support all trinkets equally. So please make sure to double check information with your theorycrafters resources, because their answers are worked out specifically for your spec. I'm simply running a lot of gear comparisons without in-depth spec knowledge.

Your own equipment, legendaries and talents quite possibly would change the order of trinkets for your character. Therefore: simming different trinkets against each other is the best way to be sure. The easiest way to do so is by heading to RAIDBOTS which uses SimulationCraft for you. Try it out.

Until I push an update the legendary trinket is simmed with itemlevel 970 but is shown with itemlevel 940.

Used settings

  • fight_style: patchwerk
  • fight_length: 300sec
  • target_error: 0.1
  • base profile: T20M
  • ptr: false
