Stacked trinket charts

This site provides an overview for all available trinkets for all specs in World of Warcraft. Please use the trinket analysis of your specs theory crafters for more accuracy.

To generate the data I'm using SimulationCraft and its standard profiles. All limitations applying to SimulationCraft apply to these graphs as well. Your own equipment, legendaries and talents change the order of trinkets for your character. Therefore: simming different trinkets against each other is the best way to know which performs best. The easiest way to do so is by heading to RAIDBOTS which uses SimulationCraft for you. Try it out.

Used settings

  • fight_style: patchwerk
  • fight_length: 300sec
  • target_error: 0.1
  • base profiles: T20M (no special profiles like DfA or gambling were used)
  • ptr: false
  • no second trinket was used for the calculation

I'm going to add more fight styles later (I plan on adding at least one multi target fight style).
