Stacked trinket charts

This site provides an overview for all available trinkets for all specs in World of Warcraft. Please use the trinket analysis of your spec theorycrafters for more accuracy.

To generate the data, I'm using SimulationCraft and its standard profiles. All limitations applying to SimulationCraft apply to these graphs as well. Your own equipment, legendaries and talents change the order of trinkets for your character. Therefore, simming different trinkets against each other is the most effective way to know which performs best. The easiest way to do so is by heading to RAIDBOTS, which uses SimulationCraft for you. Try it out.

Used settings

  • fight_style: patchwerk and beastlord
  • fight_length: 300sec
  • target_error: 0.1% (patchwerk) and 0.25% (beastlord)
  • ptr: false
  • base profiles: T20M (no special profiles like DfA or gambling were used)
  • a versatility stat stick at ilevel 880 was used in conjunction with the ones in the graph

I might add tank specs in the near future. But still only dps charts and therefore dps trinkets. As tank specs are supported quite differently, beware of strange results because of that.
