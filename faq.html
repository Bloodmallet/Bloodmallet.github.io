Are the charts up to date?

The struggle is real. Please check the time stamp at the top of each chart. I'm trying my best to keep all charts as up to date as possible. But as long as I have to use my own computer to run all simulations, this blocks all other actions on that machine for at least 24 hours. With enough support via Patreon I might be able to run these simulations in the cloud, allowing a better update circle.

Can I trust your charts?

Yes-ish - for a rough overview. Your own equipment, traits and talents change the value of items and traits for your character. Therefore, simming your own character is the most effective way to know which performs best. The easiest way to do so is by heading to RAIDBOTS, which uses SimulationCraft for you.

Why is X missing?

Chances are, I simply missed it. Please report that via github.
